For every takedown, there is a set up. For example, the arm drag is not a take-down by itself, but rather, a set up for several other takedowns.

The arm-drag can be used to set up:

     
  1. A single leg
  2. A double leg
  3. A body lock
  4. A Russian tie
  5. A trip
